# Visual Coaching Library

Bilingual EN-VI tennis coaching flipbook by **Henry Phạm Đức**.

## What it is

- **46 cards** across **11 topics**: Forehand, Backhand, Serve, Volley, Footwork, Grip, Biomechanics, Strategy, Multi-topic, Forehand V2, Volley V2
- **5 practical instructions per card** (in English, with Vietnamese translation under each)
- **230 EN instructions + 230 VI translations = 460 paragraphs** of content
- **2 editions**: Master (all 46 cards in one flipbook) and per-topic (11 individual flipbooks)
